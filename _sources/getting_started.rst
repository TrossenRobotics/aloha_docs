===============
Getting Started
===============

.. note::

  If a laptop was included in your ALOHA order, its username and password are both ``aloha``.
  It is recommended to change the password as soon as you log in.
  See the `Ubuntu documentation`_ for details on this process.

.. _`Ubuntu documentation`: https://help.ubuntu.com/stable/ubuntu-help/user-changepassword.html
